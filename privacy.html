Overview

HotTea is a Chrome browser extension committed to protecting user privacy. This policy explains how we handle user data.

Data Collection

Data We Collect

  • Google Account Information: Email address and display name (via Google OAuth)
  • Web Content: Current news page content read temporarily only when you click the extension button
  • Usage Statistics: Daily usage count (stored locally)

Data We Do NOT Collect

  • Browsing history
  • Passwords or sensitive credentials
  • Personal communications
  • Location information
  • Financial information

Data Usage

Collected data is used only for:

  1. Authentication and user identification
  2. AI conversation generation (news content transformation)
  3. Usage quota management

Data Storage

  • Local Storage: User settings and states are stored in the browser locally
  • Temporary Processing: News content is processed temporarily only during conversation generation, never permanently stored
  • No Server Storage: We do not store any personal data on our servers

Data Sharing

  • We do NOT sell, trade, or transfer your personal data to third parties
  • News content is only sent to Google AI (Gemini) API for processing
  • All API communications use encrypted connections

Data Security

  • Uses Google OAuth 2.0 for secure authentication
  • Does not store passwords or sensitive credentials
  • Follows Chrome extension security best practices

User Rights

You have the right to:

  • Sign out and clear local data at any time
  • Stop using the extension
  • Request information about how we use your data

Children's Privacy

HotTea is not designed for children under 13, and we do not knowingly collect personal information from children.

Policy Changes

If there are significant changes, we will notify users through the extension.
